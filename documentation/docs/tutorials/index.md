# Tutorials

Step-by-step tutorials for learning ORMDB concepts.

## Core Tutorials

| Tutorial | Description | Time |
|----------|-------------|------|
| [Schema Design](schema-design.md) | Define entities, fields, and relations | 15 min |
| [Querying Data](querying-data.md) | Build queries with filters and includes | 20 min |
| [Mutations](mutations.md) | Insert, update, and delete operations | 10 min |
| [Relations](relations.md) | Work with one-to-one, one-to-many, and many-to-many | 15 min |
| [Filtering](filtering.md) | Advanced filtering with AND/OR/NOT | 10 min |
| [Transactions](transactions.md) | ACID transactions for data integrity | 10 min |

## Learning Path

If you're new to ORMDB, we recommend following these tutorials in order:

1. **[Schema Design](schema-design.md)** - Start by understanding how to model your data
2. **[Querying Data](querying-data.md)** - Learn the query API for reading data
3. **[Mutations](mutations.md)** - Understand how to write data
4. **[Relations](relations.md)** - Master relational data modeling
5. **[Filtering](filtering.md)** - Build complex queries
6. **[Transactions](transactions.md)** - Ensure data consistency

## Prerequisites

Before starting these tutorials, make sure you have:

- Completed the [Getting Started](../getting-started/index.md) guide
- A running ORMDB server
- Your preferred client library installed
